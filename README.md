# Portfolio | Muhamad Tamam

Website portfolio pribadi full-stack.  
🔗 **Live Demo**: [tamam-portfolio.surge.sh](https://tamam-portfolio.surge.sh)

---

## ✨ Fitur

- ✅ Navbar responsive + hamburger menu
- ✅ Hero section dengan animasi
- ✅ Skills dengan progress bar animasi
- ✅ Projects card dengan hover effect
- ✅ Contact form (React + EmailJS)
- ✅ Mobile friendly
- ✅ Backend API sendiri (Node.js + Express)

---

## 🛠️ Tech Stack

| **Frontend** | **Backend** |
|--------------|-------------|
| HTML, CSS, JavaScript | Node.js |
| React (via Babel) | Express |
| Font Awesome | Nodemailer |
| Google Fonts | dotenv |
